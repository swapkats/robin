# DynamoDB Templates

These templates provide common single-table design patterns for various application types.

## Available Templates

### 1. Basic User/Content
- Users
- Posts/Articles
- Comments
- Likes

### 2. SaaS Multi-Tenant
- Organizations
- Teams
- Members
- Roles
- User-Org relationships

### 3. E-commerce
- Products
- Orders
- Cart items
- Customers
- Reviews

### 4. Social Network
- Users
- Posts
- Followers/Following
- Messages
- Notifications

## Template Structure

Each template includes:
- Entity definitions
- Primary key patterns
- GSI patterns
- Access pattern examples
- TypeScript types
- Query helper functions

## Usage

Robin uses these templates when designing DynamoDB schemas. The templates serve as starting points that are customized based on specific application requirements.
